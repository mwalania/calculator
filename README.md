# Calculator 
A functional mathematical computer that allows user to interact via the calculator buttons. The app does not evaluate more than a single pair of numbers at a time.

## Installation
1. Navigate your repos directory i.e where you will clone the repo
```bash
cd ~/repos/
```
2. Clone the github repository.
```bash
git clone 'git@github.com:mwalania/calculator.git'
```
3. Navigate to working directory
```bash
cd ~/repos/calculator
```
4. Open VS Code
```bash
code .
```
5. Creating your files e.g
```bash
touch index.html style.css script.js
```
